# Campus Event Reminder and Recommender

# Overview
This project helps students manage and discover campus events.

Users can:
- Add events
- View events
- Get recommendations based on interests
- Rate events
- View the top rated event

# Technologies Used
Python

# Challenge
One challenge I faced was creating recommendation logic and allowing users to rate events.

# What I Learned
I learned how to use lists, dictionaries, loops, functions, conditionals, and user input.

# Future Improvements
If I had more time, I would add notifications and event sorting by date.
